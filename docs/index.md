# GCP 文档索引（用户 / SDK / 二次开发）

本目录聚焦如何使用 GCP，而不是其内部设计推导细节。

## 快速开始

- `quickstart.md`：5 分钟从解析、推导到执行跑通一段 Outline 代码。

## SDK 参考

- `sdk-reference.md`：完整 API 参考（解析、AST/ASF、推导、执行、插件、错误、元数据）。
- `dot-completion.md`：Dot Completion 在 IDE 与 LLM 工具链中的统一调用路径与约束。

## 论文与研究资料

- `paper/papers.md`：论文分工、差异矩阵与推荐阅读路径。

## 使用建议

- 首次接入先读 `quickstart.md`。
- 需要查 API 参数、返回值、调用顺序时查 `sdk-reference.md`。
- 需要做补全、hover、成员推导时查 `dot-completion.md`。
